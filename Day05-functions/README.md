# Day 5 - Functions

Today I learned how to organize code using functions in Dart.

At first, I was just repeating print statements, but now I can group code into reusable blocks which makes everything cleaner and easier to manage.

## Files

- welcome.dart
- print_name.dart
- print_line.dart
- simple_profile.dart

## What I Learned

- How to create a function using `void`
- How to call a function inside `main()`
- Why functions help avoid repeating code
- How to structure a program using multiple functions

## What I Practiced

- Created a welcome function
- Printed names using reusable functions
- Built simple UI-style output using functions
- Split a program into header, body, and footer sections

## What I Found Easy

Calling functions was simple once I understood that code only runs when you call it inside `main()`.

## What Was Slightly Confusing

At first, I kept forgetting that defining a function is not the same as executing it.

## Key Takeaway

Functions help turn messy repeated code into organized blocks that can be reused anytime.
