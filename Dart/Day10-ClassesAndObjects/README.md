# Day 10 - Classes & Objects

Today I started learning Object-Oriented Programming (OOP) in Dart.

The biggest thing I learned is the difference between a class and an object. 

- A class is a blueprint, while an object is an actual instance created from that blueprint.

This is the foundation of Flutter because almost everything in Flutter is an object created from a class.

## What I Found Challenging

At first, I was a little confused about the difference between a class and an object. 

-After practicing, I understood that one class can create many different objects,-
 and each object stores its own data independently.

## Key Takeaways

- A class is a blueprint.
- An object is created from a class.
- Each object has its own data.
- Multiple objects can be created from the same class.
- Classes are the foundation of Flutter development.
