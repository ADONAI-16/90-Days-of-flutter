# Dart Programming Journey 🚀

Welcome to my Dart learning journey as part of my **90 Days of Flutter Challenge**.

Before building Flutter applications, I am strengthening my foundation in Dart by learning its core concepts, object-oriented programming, and modern language features.

## 📌 About Dart

Dart is a modern, object-oriented programming language developed by Google. It is the primary language used for Flutter development, allowing developers to build fast and scalable applications for mobile, web, and desktop platforms.

## 🎯 Learning Goals

- Understand Dart syntax and programming fundamentals
- Master object-oriented programming concepts
- Learn how Dart handles collections and data structures
- Practice writing clean and reusable code
- Build a strong foundation for Flutter development
