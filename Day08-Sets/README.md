# Day8- Sets

Today I learned about Sets in Dart. Unlike Lists, Sets only store unique values, so duplicate items are automatically ignored.

I also learned that Sets don't use indexes because they are unordered collections.

## What I Found Challenging

The biggest difference from Lists was understanding that Sets don't have indexes because they are unordered. I also learned how `union()`, `intersection()`, and `difference()` compare two Sets in different ways.

## Key Takeaways

- Sets automatically remove duplicate values.
- Use a Set when uniqueness is more important than order.
- Set operations make comparing collections much easier.
- `for-in` is the best way to loop through a Set.
