# Day 3 - Conditional Statements

Today I learned how to make decisions in Dart using:

* if
* else
* else if
* Nested if statements

## Files

* age_checker.dart
* exam_grade.dart
* login_validation.dart
* bank_withdrawal.dart
* student_result.dart

## What I Practiced

### Age Checker

Used conditions to determine whether someone is an adult or a minor.

### Exam Grade

Assigned grades based on a student's score.

### Login Validation

Checked whether login information was valid before allowing access.

### Bank Withdrawal

Verified whether an account had enough balance before withdrawing money.

### Student Result

Combined multiple conditions to calculate and display a student's grade.

## Key Things I Learned

* Conditions evaluate to either `true` or `false`.
* `if` runs code when a condition is true.
* `else` runs when the condition is false.
* `else if` helps handle multiple cases.
* The order of conditions matters.
* Conditions are used everywhere in real applications.
